# 体检项目推荐技能

> 让每一次体检推荐，都成为客户信任的开始。

## 流程

```
[触发] 询问体检 → 信息收集 → 循证推荐 → 发送二维码预约
```

## 第一步：信息收集

询问顺序（每次1-2个问题）：
1. "给自己还是家人？"
2. "年龄和性别？"
3. "有没有特别想检查的部位或症状？"
4. "家族有没有心脑血管/肿瘤/糖尿病病史？"
5. "之前体检有没有已知异常？"

## 第二步：循证推荐

⚠️ **重要约束：推荐项目必须严格在 checkup_items.md 清单范围内**
- 只推荐清单中有的项目，不能额外创造
- 如需加项，先确认清单中是否有对应编码
- 清单范围外项目不推荐，建议客户咨询医疗机构

### 风险排名（按年龄+性别）

**男性**
| 年龄 | Top 1-3 |
|-----|---------|
| 18-35 | 肝癌、甲状腺癌、肺癌 |
| 36-49 | 肝癌、肺癌、心脑血管 |
| 50-64 | **肺癌、心脑血管**、肝癌 |
| 65+ | **肺癌、心脑血管**、胃癌 |

**女性**
| 年龄 | Top 1-3 |
|-----|---------|
| 18-35 | **乳腺癌**、甲状腺癌、宫颈癌 |
| 36-49 | **乳腺癌**、甲状腺癌、肺癌 |
| 50-64 | **乳腺癌**、肺癌、心脑血管 |
| 65+ | **肺癌、心脑血管**、结直肠癌 |

### 加项规则

| 情况 | 加项 | 依据 |
|-----|------|-----|
| 50岁+男性 | 低剂量螺旋CT | 肺癌发病率第一 |
| 50岁+男性 | 胃镜/肠镜 | 消化道肿瘤高发 |
| 男性 | 前列腺特异抗原 | 前列腺癌筛查 |
| 40岁+女性 | 乳腺彩超+钼靶 | 乳腺癌发病率第一 |
| 女性 | TCT+HPV | 宫颈癌筛查 |
| 心脑血管症状 | 心电图、心脏彩超、运动心电图 | 心脑血管事件高发 |
| 胃部不适/腹泻 | 胃镜(HLZXX0205)、便常规+隐血 | 胃癌早筛、消化道出血排查 |
| 头痛/头晕 | 头颅CT/ MRI、TCD | 脑血管评估 |
| 便秘/便血 | 直乙肠镜(HLZXX0259)、便隐血定量 | 结直肠癌筛查 |
| 肿瘤家族史 | 全套肿瘤标志物、基因检测 | 早筛早诊 |
| 吸烟/粉尘 | 低剂量螺旋CT、肺功能 | 肺部深度 |

## 第三步：推荐输出

格式：
```
【风险评估】55岁男性：高发风险为 肺癌 > 心脑血管 > 肝癌

【基础套餐】高客最小项（已含30+项）
- 血尿便常规、肝肾功能、血糖血脂
- 心电图、颈动脉彩超、动脉硬化
- 腹部彩超、甲状腺彩超、甲功
- 肿瘤标志物、骨密度、肺功能

【循证加项】
🔴 低剂量螺旋CT — 肺癌是55岁男性发病率首位
🔴 胃镜 — 胃癌风险进入Top 3
🟡 前列腺特异抗原 — 男性50+建议排查
```
