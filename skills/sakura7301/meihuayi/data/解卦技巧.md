# 梅花易数 · 解卦技巧

> 体用生克、月令旺相、断卦方法论

**AI 执行流程**

1. **收集信息**：确认用户想问的问题
2. **确认起卦方式**：时间起卦（默认）或数字起卦
3. **调用排盘程序**：执行 meihuayi.py，获取排盘结果
4. **按解卦技巧.md 的断卦步骤逐步分析**
5. **输出格式**：先展示排盘结果（原始文本），再给出推断
6. **语气要求**：严肃但不神秘，理性分析，末尾提醒仅供参考
7. **问题参数处理**：
  - 若用户直接给出问题，AI 提取并传给排盘程序
  - 若用户没有明确问题，AI 主动询问："请你具体说一下想问的事情"
  - 排盘程序会把问题显示在输出第一行，方便对照

## 一、月令旺相（五行旺衰）

> 五行在十二个月中的旺衰状态，是断卦的重要依据

### 旺相休囚死释义

- **旺**：最旺盛，当令之气（同我者旺）
- **相**：次旺，受旺气所生（我生者相）
- **休**：休息，生我者休
- **囚**：被困，克我者囚
- **死**：最衰，我克者死

## 二、体用生克

### 基础概念

- **体卦**：主卦中**无动爻**的经卦，代表求测者、本人、主体
- **用卦**：主卦中**动爻所在**的经卦，代表所问之事、对方、环境

> 例如：天地否，初爻动，初爻在下卦坤，则下卦坤为用卦，上卦乾为体卦

### 生克关系

| 关系 | 名称 | 吉凶 | 解释 |
|------|------|------|------|
| 体克用 | 财 | 吉 | 以少胜多，主导权在握 |
| 用克体 | 官鬼 | 凶 | 祸害临头，压力巨大 |
| 体生用 | 泄气 | 耗 | 劳心费力，付出多回报少 |
| 用生体 | 生气 | 吉 | 外来助力，事半功倍 |
| 体用比和 | 比和 | 吉 | 和谐顺利 |

### 动爻的影响

梅花易数**一卦一爻动**，动爻发生变化产生变卦，需要综合分析本卦、互卦、变卦三者。

### 互卦的作用
互卦代表事情发展的中间过程、内在动因，也代表求测者内心的真实状态。
互卦生体 → 过程对自己有利
互卦克体 → 过程中阻力重重
互卦与体用五行一致 → 过程顺畅

### 变卦的作用
变卦代表事情的最终走向和结果。
- 变卦生本卦体 → 结局好转
- 变卦克本卦体 → 结局受损
- 变卦与本卦体同五行 → 结局稳定
注：变卦中的体卦与本卦相同，一般只看变卦的用卦（即动爻变化后的那一卦）对本卦体的影响。

## 三、断卦步骤

### 标准流程

```
第1步：调用排盘程序得到排盘结果(执行meihuayi.py)
  ↓
第2步：结合体用生克，推断基本的吉凶
  ↓
第3步：结合万物类象取象，将所闻之事对应到卦上(参照`附录：事项快速取象索引表`并在`./data/万物类象.md`中查阅需要的内容)
  ↓
第4步：结合三要十应，进一步补充卦和实际情况的联系(这一步需要向用户主动询问，从而获得外应，如用户表示没有值得注意的信息，则跳过外应，得到信息后，结合`./data/三要十应.md`进一步推断)
  ↓
第5步：整合推断结果(将体用生克、月令旺衰、万物类象、外应四个维度的结论汇总，用简洁的语言给出：① 核心结论（吉/凶/平）② 具体分析（事情走向、注意事项）③ 行动建议)
  ↓
第6步：推断应期（如需要）
```

## 四、断卦要诀

梅花排盘出来之后，看起来有六个经卦，但是变卦中的体卦，我们一般不看(因为体卦和本卦一样，没有任何参考意义)，所以实际上是五个经卦，即主卦的上下两卦，互卦的上下两卦，产生动爻而得到的变卦。因而，一切的五行生克都是对于本卦而言的，“非彼无我，非我无所取”，记住这段话，所有其它的卦都是作用与本卦的，其生克关系也是作用于本卦的。

### 1. 先看体用
> 体用生克是根本

- 体克用 → 吉
- 用克体 → 凶
- 体生用 → 耗
- 用生体 → 吉
- 体用比和 → 吉

### 2. 次看旺衰
> 月令掌生杀

无论体用：
- 卦在月令旺相 → 力量强
- 卦在月令休囚 → 力量弱
- 卦被月令所克 → 力量极弱
- 同五行的卦多未必就真的旺，也未必真的衰：被月令所生，再弱也有一战之力；被月令所克，再强也无从下手。
- 物极必反：这是古典思维，当一种事物旺到极致时反而会走下坡路，反之亦然，当一种事物衰到极致时也会慢慢变好。
- 五行相克，未必克的动：比如我现在变卦为金，克体卦的木，但是现在寅月，体卦得月令，木极旺，那么此时的金不仅克不动，还反被金侮。

### 3. 结合万物类象
> 象数合一，取象准确

- 根据所问之事选取合适象意
- 卦象+象意=具体事件

### 4. 参看三要十应
> 内外感应，天人合一

- 什么时候需要外应：当卦本身难以决断，捉摸不定时，需要取外应。
- 怎么取外应：主动询问用户，让用户简单描述一下听到的、看到的、感觉到的任何事物，将其与卦联合起来，我举个例子，当我在推算某人的吉凶时，恰巧看到一个婴儿哭泣，婴儿哭泣是担惊受怕之象，那么这个外应表达的就是凶象。
- 外应与卦象相符 → 更准
- 外应与卦象相悖 → 需细辨

## 五、应期推断

### 基本方法

> **成卦之数 + 时数 = 应期**

计算公式：
- 成卦之数 = 上卦先天数 + 下卦先天数（1-8 + 1-8 = 2-16）
- 时数 = 时辰地支序数（子=1, 丑=2, ... 亥=12）
- **应期数字范围：3-28**

### 应期断法核心

> **必须根据所问之事的性质，结合现实逻辑来推断时间单位！**

不是数字大就是年，数字小就是日。要看问的是什么：

| 问的事情 | 时间单位推断示例 |
|----------|------------------|
| 外卖/快递 | 分钟（13分钟） |
| 今天的 事 | 时辰/小时 |
| 本月的事 | 日/旬 |
| 今年的事 | 月 |
| 长期的事 | 年 |

### 断应期要点

1. **先问什么**：问今天的事，按日计；问长远的事，按年计
2. **灵活判断**：数字是参考，关键符合现实逻辑
3. **参考月令**：旺相之时应期近，休囚之时应期远
4. **不固守**：有的卦无需应期，切勿强求

### 示例

```
例1：问外卖何时到
    天地否卦，上卦乾1 + 下卦坤8 = 9
    丑时（2），应期=9+2=11
    断：11分钟内送到

例2：问今日财运
    数字11，巳时（6），应期=17
    断：约17天内有财运

例3：问今年工作
    数字15，亥时（12），应期=27
    断：约27个月后（约2年后）有转机
```

---

*数字只是参考，关键看问什么事。问外卖送到，13就是13分钟；问升职加薪，13可能就是13个月。*

## 七、案例示范

**案例**：问近期工作是否有晋升机会

---

### 排盘结果

```
起卦：时间起卦（假设现在三月辰月，上午巳时）
本卦：水火既济（上卦坎☵，下卦离☲）
互卦：互见雷风（震☳ / 巽☴）
变卦：水山蹇（上卦坎☵，下卦艮☶）
动爻：四爻动（四爻在上卦坎，故坎为用，离为体）
```

---

### 第1步：体用生克

- **体卦**：离（火）→ 代表求测者本人
- **用卦**：坎（水）→ 代表晋升这件事

> 水克火 → **用克体 → 官鬼 → 凶**

基本判断：晋升这件事对求测者形成压制，事情主动权不在自己手里，阻力较大。

---

### 第2步：月令旺衰

三月辰月，查表：

| 五行 | 状态 |
|------|------|
| 木 | 囚 |
| 火 | 休 |
| 土 | 旺 |
| 金 | 相 |
| **水** | **死** |

- **体卦离（火）：休**，本就不强
- **用卦坎（水）：死**，月令辰土克水，水极衰

关键转折就在这里：

> 用克体本来是凶，但用卦坎水在辰月为**死**，克不动体卦。体卦离火虽然休，但至少比用卦强。此消彼长，用虽然想克体，却力不从心。

---

### 第3步：万物类象取象

结合所问之事（晋升/工作）：

- **离卦**（体）：光明、文书、考核、表现——求测者本身有才华，只是时机未到
- **坎卦**（用）：险阻、暗流、竞争者、上司的态度——晋升之路有隐形障碍，可能有人暗中阻拦
- **互卦震巽**：震为行动、巽为沟通——中间过程需要主动出击，多沟通斡旋
- **变卦蹇（水山艮）**：蹇者，难也，山水相阻——结果显示前路仍有阻碍，不可操之过急

---

### 第4步：三要十应

此卦用克体但用卦极衰，结论尚不明朗，需要取外应辅助判断。

**主动询问用户**：

> "卦象有些两可，请你回忆一下起卦前后，听到了什么声音，或者看到了什么特别的东西，哪怕一闪而过也行。"

假设用户回答："起卦前听到有人在笑，然后看到窗外有阳光。"

- **笑声**：吉声，喜事将近
- **阳光**：光明之象，对应离卦（体），体得外应支撑

外应与卦象方向一致：**体虽弱但有外应扶持，更偏吉**。

---

### 第5步：综合判断

逐层梳理：

| 维度 | 结论 |
|------|------|
| 体用生克 | 用克体，本凶 |
| 月令旺衰 | 用卦坎水辰月为死，克不动体 |
| 互卦 | 震巽主沟通行动，中间过程需努力 |
| 变卦 | 蹇卦，阻滞，结果不顺畅 |
| 外应 | 笑声+阳光，吉象，扶持体卦 |

**综合断语**：

> 近期晋升阻力确实存在，背后可能有竞争者或上司态度不明朗。但阻力并不足以压制你，时机虽不成熟，并非全无希望。当下最忌被动等待，互卦震巽提示需要主动沟通、积极表现。变卦蹇提示此事不会一步到位，需要耐心。结论：**短期内晋升可能性不大，但半年至一年内若主动作为，仍有机会。**

---

### 第6步：应期

```
上卦坎（6）+ 下卦离（3）= 9
巳时（6）
应期 = 9 + 6 = 15
```

问的是今年工作，时间单位取**月**：

> **约15个月后**，晋升机会较为明朗。

---

包括三部分：
- 问题
- 排盘结果
- 推断
