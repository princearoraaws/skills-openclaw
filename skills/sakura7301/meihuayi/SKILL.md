# 梅花易数卜卦 ☯️

中华传统占卜术数，源于宋代邵康节。

## 功能

- **时间起卦** - 根据年月日时起卦
- **数字起卦** - 任意三个数字起卦
- **自动解卦** - 本卦+互卦+变卦，体用生克，月令旺衰

## 安装使用

```bash
# 克隆或复制 skill 到本地
cd 所在目录

# 排盘
cd meihuayi
python3 meihuayi.py [命令] [参数]
```

## 命令行

```bash
# 时间起卦（使用当前时间）
# python3 meihuayi.py time [用户的问题]
# 示例
python3 meihuayi.py time 今天出行是否顺利

# 数字起卦(必须是连续的三个数字)
# python3 meihuayi.py num <三位数> [用户的问题]
# 示例
python3 meihuayi.py num 258 近期能否升职加薪
```

### 推断

**体用生克分析**：
- 体卦：艮（土）
- 用卦：风（木）
- 关系：体生用（泄气）→ 示意工作需要付出努力

**月令旺衰**：
- 辰月：木囚、火休、土旺、金相、水死
- 解读：体卦艮土旺相，力量充足；用卦风木被克制，事情需要时间推进

**万物类象**（参考工作事项表）：
- 艮卦：阻碍、稳定、守成
- 风卦：流动、改变、扩展
- 综合：虽有阻力，但稳步可行

**三要十应**：
起卦时听到笑声 → 吉应 → 与卦象相符

**综合断语**：
此次工作调整虽然需要付出努力（体生用），但基础稳定（土旺）。建议积极主动（风木需突破克制），预期三个月内有明显进展。

**应期**：
艮7 + 风5 = 12，午时6，应期 = 18，取月计 → 18个月后见分晓（但实际应该按现实逻辑判断，如询问近期，则按日或周计）
