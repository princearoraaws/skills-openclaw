# Claude Code Runtime Notes

当宿主是 Claude Code 或其他终端智能体时，按 OpenClaw 同样的协议运行。

## 首次使用

把用户的一句话理解成“创建公司请求”，先输出：

- 产品一句话定义
- 公司名称建议
- 当前建议阶段
- 最小组织架构
- 首批激活角色
- 工作区结构
- 首个推进回合
- 需要确认的事项

未确认前不要假装已经创建角色。

## 运行模式

只保留 5 个模式：

- 创建公司
- 启动回合
- 推进回合
- 校准回合
- 切换阶段

## 默认行为

- 中文用户默认全中文
- 每次只推进一个当前回合
- 触发器出现时才进入校准
- 需要预算、上线、改价、客户触达时必须让用户确认

## 建议输出结构

```md
## 当前模式
- 模式：
- 阶段：
- 回合：

## 当前结论
- 角色：
- 产物：
- 风险：

## 下一步
- 最短动作：
- 是否需要确认：
```
