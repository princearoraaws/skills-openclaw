# 私人研究院 · My Research Lab

> 你的 AI 研究助手——说启动就运转，自动采集 + 深度脑爆 + 直接产出，越用越懂你。

## 这是什么

**私人研究院** 是一个全自动运转的个人智库 Skill。

它为你做三件事：
1. **发现** — 自动采集你关注领域的最新信息，过滤噪音，只推高价值内容
2. **脑爆** — 把发现的内容和你的背景交叉碰撞，判断能做成什么
3. **动手** — 不只给建议，直接产出文档、网页、Skill 等可用成果

零配置即可启动，支持任意研究领域。

## 适合谁用

| 你是谁 | 你的研究院 | 典型产出 |
|--------|-----------|---------|
| 公益从业者 | AI 向善研究院 | 公益 AI 案例库、工具评测、方案文档 |
| 程序员 | 技术研究院 | 框架追踪、效率工具推荐、代码 Skill |
| 产品经理 | 产品研究院 | 竞品分析、用户研究、需求文档 |
| 设计师 | 设计研究院 | 设计趋势、灵感库、工具对比 |
| 任何人 | 你说了算 | 你定义产出 |

## 快速启动

说一句话就能启动：

```
"启动研究院"
"帮我研究 AI 公益"
"我想建一个美食研究院"
"脑爆一下这个想法"
```

院长（AI 助手）会问你 3 个问题完成初始化，然后立刻开始运转。

## 三种运转模式

```
发现日 → 采集信源 → 审稿过滤 → 推送简报
脑爆日 → 深度分析 → 交叉碰撞 → 判断能做什么
动手周 → 你说做 → 院长立刻动手 → 交付成果
```

## 特色功能

- **智能信源管理**：内置 9 大领域信源库，根据你的方向自动匹配，支持随时增删
- **7 条审稿清单**：每条推送必过质量关，不相关的、重复的、没价值的自动过滤
- **脑爆引擎**：不只分析，直接判断能做成 Skill / 网页 / 文档中的哪一种
- **自进化记忆**：记录你的偏好和反馈，下次推送更精准
- **零配置运行**：不需要任何 API Key 或外部账号也能完整运行
