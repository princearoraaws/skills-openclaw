## 3. How to use

### 调用方式

```bash

# 指定时间排盘
python3 ~/.openclaw/skills/qizheng-siyu-skill/scripts/qizheng_pan.py --date "2026-03-21 15:00"

# 指定地点（经纬度）
python3 ~/.openclaw/skills/qizheng-siyu-skill/scripts/qizheng_pan.py --date "2026-03-21 15:00" --lat 39.9 --lon 116.4
```

| --lat | | 纬度（默认 39.9） |
| --lon | | 经度（默认 116.4） |

## 7. 断卦规则

### 宫位分析

- **命宫**：主一生运势、性格、外貌
- **财帛**：主财运、收入、理财
- **夫妻**：主婚姻、感情、配偶
- **官禄**：主事业、官职、名声
- **疾厄**：主健康、疾病、灾祸
- **迁移**：主出行、变动、外地

### 星曜判断

1. **太阳**：君主之星，主事业、名声、父亲
   - 庙旺：光明磊落，事业有成
   - 落陷：韬光养晦，待时而动

2. **太阴**：后妃之星，主情感、母亲、女性
   - 庙旺：情感丰富，贵人相助
   - 落陷：情绪波动，宜静心

3. **木星**：吉星之首，主财运、扩张、智慧
   - 入财帛：财运亨通
   - 入命宫：一生富贵

4. **火星**：凶星，主行动、竞争、冲突
   - 庙旺：行动力强，宜开拓
   - 落陷：防冲动行事

5. **土星**：凶星，主压力、责任、限制
   - 庙旺：稳重踏实，能担重任
   - 落陷：压力过大，宜放松

6. **金星**：吉星，主爱情、艺术、财富
   - 庙旺：人缘好，艺术天赋
   - 落陷：感情波折

7. **水星**：中性星，主智慧、沟通、学习
   - 庙旺：聪明智慧，善于表达
   - 落陷：思虑过多
