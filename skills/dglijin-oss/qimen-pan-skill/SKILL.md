# 奇门遁甲排盘技能

## 1. Description

当用户询问奇门遁甲排盘，或需要根据时间分析运势时，调用此技能获取标准排盘数据。

**核心功能：**
- 公历/农历转换（1900-2100 年，精度±1 分钟）
- 精确节气计算（天文算法）
- 阴阳遁判断与局数计算
- 四柱排盘（年柱、月柱、日柱、时柱）
- 值符值使推算
- 九宫落位排布（八门、九星、八神）
- **v3.0.0 新增：**
  - 完整 81 格局判断（吉格/凶格）
  - 吉凶量化评分（0-100）
  - 应期推算
  - 趋吉避凶建议
- **v2.0.1 新增：**
  - 断卦解盘自动化分析
  - 用神选取规则细化
  - 吉凶判断逻辑
- 真太阳时校正（支持城市/经度）

3. **调用排盘脚本** — 获取标准排盘数据
4. **返回格式化结果** — 按标准格式输出

## 5. Edge cases

| 情况 | 处理方式 |
|------|----------|
| 时间格式不清晰 | 要求用户提供明确时间，如"2026 年 3 月 18 日 15:00" |
| 超出可排范围（1900 年以前/2100 年以后） | 告知用户时间范围限制 |
| 网络不可用 | 使用本地算法计算，提示数据可能略有偏差 |
| 时辰交界（如 14:59/15:01） | 精确到分钟，按真太阳时校正 |

## 8. 奇门基础知识

### 阴阳遁规则

| 节气范围 | 遁局 | 局数范围 |
|----------|------|----------|
| 冬至~芒种（12 月~6 月） | 阳遁 | 阳遁 1~9 局 |
| 夏至~大雪（6 月~12 月） | 阴遁 | 阴遁 1~9 局 |

### 八门顺序

```
休门 → 生门 → 伤门 → 杜门 → 景门 → 死门 → 惊门 → 开门
```

### 九星顺序

```
天蓬 → 天任 → 天冲 → 天辅 → 天英 → 天芮 → 天柱 → 天心 → 天禽
```

### 八神顺序

```
值符 → 螣蛇 → 太阴 → 六合 → 白虎 → 玄武 → 九地 → 九天
```

## 10. 断卦解盘规则详解（v2.0.1 新增）

### 10.1 用神选取规则

用神是断卦的核心，根据问事类型选取对应的门、星、神：

| 问事类型 | 用神门 | 用神星 | 用神神 | 用神干 |
|----------|--------|--------|--------|--------|
| 财运 | 生门 | - | 六合 | 戊 |
| 事业 | 开门 | - | 值符 | 庚 |
| 婚姻 | 六合 | - | 太阴 | 乙庚 |
| 健康 | 死门 | 天芮 | 白虎 | - |
| 官司 | 惊门 | - | 玄武 | 辛 |
| 出行 | 开门 | - | 九天 | 丁 |
| 求子 | 生门 | - | 九地 | 丁 |
| 失物 | 杜门 | - | 玄武 | 辛 |
| 考试 | 景门 | 天辅 | - | 丁 |
| 交易 | 生门 | - | 六合 | 戊 |

### 10.2 吉凶判断标准

#### 门吉凶
- **吉门**：开门、休门、生门
- **中门**：杜门、景门
- **凶门**：伤门、死门、惊门

#### 星吉凶
- **吉星**：天冲、天辅、天禽、天心、天任
- **中星**：天英
- **凶星**：天蓬、天芮、天柱

#### 神吉凶
- **大吉**：值符
- **吉神**：太阴、六合、九地、九天
- **凶神**：螣蛇、白虎、玄武

### 10.3 五行生克分析

宫位五行属性：
- 坎宫属水，坤宫属土，震宫属木，巽宫属木
- 中宫属土，乾宫属金，兑宫属金，艮宫属土，离宫属火

五行生克关系：
- 相生：木→火→土→金→水→木
- 相克：木→土→水→火→金→木

**判断规则：**
- 用神宫生问事宫：吉，事易成
- 用神宫克问事宫：凶，事多阻
- 问事宫生用神宫：泄气，需付出
- 问事宫克用神宫：费力，难成
- 比和：平顺，事可成

### 10.4 特殊格局判断

| 格局 | 特征 | 吉凶 |
|------|------|------|
| 伏吟 | 门星落本宫 | 主慢、主静 |
| 反吟 | 门星落对冲宫 | 主快、主动 |
| 五不遇时 | 时干克日干 | 凶 |
| 三奇得使 | 乙丙丁 + 吉门 | 大吉 |
| 玉女守门 | 丁奇 + 休门 | 吉 |
| 天遁 | 丙 + 生门 + 九天 | 大吉 |
| 地遁 | 乙 + 开门 + 九地 | 吉 |
| 人遁 | 丁 + 休门 + 太阴 | 吉 |

### 10.5 断卦流程

1. **定用神** — 根据问事类型选取对应门、星、神
2. **查落宫** — 找出用神在九宫中的位置
3. **判吉凶** — 根据门、星、神的吉凶属性判断
4. **析生克** — 分析用神宫与问事宫的五行生克
5. **看格局** — 检查是否有特殊格局
6. **下断语** — 综合以上信息给出断语和建议
