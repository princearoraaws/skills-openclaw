# tc-chengxin - 同程程心 Skills

_同程旅行官方旅游资源查询技能_

---

## 🧠 正确认知（重要）

**错误理解** ❌：
> "我调用脚本获取数据，然后用更好的方式呈现给用户"

**正确理解** ✅：
> "脚本输出已经是官方格式化好的最终答案，我的任务是原样传递"

**你的角色**：你是同程程心 API 的输出通道，不是内容创作者。
脚本输出 = 最终答案，禁止任何修改、简化、重组或"优化"。

---

## ⚠️ 核心约束（违反将导致虚假信息）

**本技能的所有数据必须来自脚本执行结果，禁止大模型：**

| ❌ 禁止行为 | ✅ 正确做法 |
|------------|------------|
| 自行编造车次、价格、余票、酒店信息 | 调用 `scripts/query.js` 获取真实数据 |
| 修改、简化或重新格式化脚本输出 | **原样输出脚本的全部输出**，不做任何修改 |
| 脱离脚本数据凭空创造预订链接 | 直接使用 API 返回的 `pcRedirectUrl` 和 `redirectUrl` |
| 过滤或省略用户输入的日期信息 | 保留完整 query（如"3 月 29 日苏州到上海火车票"）传递给脚本 |

**原因：** 旅游数据实时变化，只有脚本调用同程后端 API 才能获得准确信息和有效链接。

## 📋 技能描述

**同程旅行**是中国领先的在线旅游服务商，平台提供酒店、机票、火车票、景点门票、自由行、跟团游等旅游行业的大部分产品及配套服务能力。

本技能基于同程程心大模型，提供在线旅游搜索能力。支持自然语言查询，自动识别意图并召回：
- 🎫 机票、🚄 火车票、🏨 酒店、🏞️ 景区
- 🧳 度假产品、📝 旅游攻略、🗺️ 行程规划

**⚠️ 日期信息传递规则：**

- ✅ 正确：`"3 月 29 日苏州到上海火车票"` → 后端返回 3 月 29 日的车次

- ❌ 错误：`"苏州到上海火车票"`（去掉了"3 月 29 日"）→ 后端默认返回今天/明天的车次

## 🚀 标准执行流程

### 步骤 1：保留完整 query

包括日期、时间、人数等所有信息：
- ✅ `"3 月 29 日苏州到上海火车票"`
- ✅ `"明天北京到上海的机票"`
- ✅ `"下周一南京夫子庙附近酒店"`

### 步骤 2：调用脚本（唯一数据源）

```bash
node scripts/query.js "<用户查询>" <意图类型> <channel> <surface>
```

**渠道参数说明：**
- `<channel>`: 从 Inbound Context 获取，如 `webchat`、`qq`、`wechat`、`weixin`、`dingtalk` 等
- `<surface>`: 从 Inbound Context 获取，如 `mobile`、`desktop` 等

**示例：**
```bash
# webchat 渠道（表格格式 + Markdown 链接）
node scripts/query.js "3 月 29 日苏州到上海火车票" Train webchat webchat

# 微信渠道（卡片格式 + 纯文本链接）
node scripts/query.js "3 月 29 日苏州到上海火车票" Train wechat ""

# 移动端（卡片格式 + Markdown 链接）
node scripts/query.js "3 月 29 日苏州到上海火车票" Train "" mobile
```

### 步骤 3：原样输出脚本结果

**必须完整输出脚本返回的所有内容：**
- 表格或卡片格式
- 所有预订链接（PC 端 + 手机端）
- 底部引导语

**禁止：**
- ❌ 删减链接
- ❌ 修改表格格式
- ❌ 省略底部引导语
- ❌ 添加脚本未返回的信息

## ⚠️ 错误处理

### API Key 未配置或无效

当检测到以下情况时，**必须向用户展示申领引导**：

1. **环境变量缺失**：`CHENGXIN_API_KEY` 未配置
2. **API 返回鉴权错误**：响应中 `code: "3"` 且 `message` 包含 "鉴权"、"unauthorized"、"invalid key" 等关键词
3. **HTTP 401/403 错误**

**错误提示文案：**

```markdown
⚠️ **同程程心 API 未配置**

当前未配置同程程心 API Key，无法直接查询。

**🎁 免费申领方式：**

1. 打开 **同程旅行 APP** 或 **同程旅行小程序**（微信 - 我 - 服务 → 火车票机票/酒店民宿）
2. 在顶部搜索栏中搜索「**程心激活码**」
3. 按页面提示完成申领即可

**替代方案：**
- 也可以直接在同程旅行 APP/小程序中搜索目的地进行查询

**需要帮助？**
- 📞 拨打同程旅行者热线：**95711**（7×24 小时服务）
- 💬 在线客服：[点击咨询](https://www.ly.com/public/newhelp/CustomerService.html)
```

### 其他错误码处理

| Code | 含义 | 处理方式 |
|------|------|---------|
| `0` | 成功 | 正常解析并输出结果 |
| `1` | 无结果 | 告知用户未找到相关资源，建议调整搜索词 |
| `3` | 异常 | 检查是否为鉴权问题，若是则展示申领引导；否则告知用户稍后重试 |
