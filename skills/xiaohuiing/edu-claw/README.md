# 小英老师 - 主动式英语学习家教

一个为 6-12 岁孩子设计的英语 AI 老师，**不是冷冰冰的答题机器，是真正有温度的老师**。

## ✨ 核心亮点

| 功能 | 说明 |
|------|------|
| 🎭 **拟人化老师** | 记住孩子名字、喜好，像真正的老师一样交流 |
| 😊 **情绪承接** | 接住孩子情绪，永远鼓励为主 |
| 🎮 **兴趣引导** | 用孩子喜欢的内容教学（游戏/足球/卡通） |
| 📝 **造句练习** | 不仅是答题，是完整的教学流程 |
| 📊 **智能评分** | 从创意、词汇、语法三维度评分 |
| 🏆 **积分激励** | 学习攒积分，培养学习习惯 |
| 🎁 **奖励系统** | 孩子许愿 → 家长确认 → 达成目标 |

---

## 📖 功能详解

### 1. 拟人化交互

**不是问答，是师生对话**

- 记住孩子名字、年级
- 记住孩子喜好（游戏/足球/卡通）
- 首次见面有"仪式感"
- 每次对话像真正的师生交流

### 2. 兴趣引导教学 🎮⚽📺

**用孩子喜欢的内容教学**

如果孩子喜欢游戏：
> "会用 Double Kill! First Blood! 教你英语！"

如果孩子喜欢足球：
> "会用 Goal! Penalty! 教你英语！"

如果孩子喜欢卡通：
> "会用孩子喜欢的卡通人物台词教学！"

### 3. 智能评分

从三个维度打分：
| 维度 | 分值 |
|------|------|
| 内容创意 | 0-10分 |
| 词汇使用 | 0-10分 |
| 语法正确 | 0-10分 |

### 4. 积分激励系统

| 行为 | 积分 |
|------|------|
| 完成练习 | +1 ~ +3 |
| 连续学习3天 | +5 |
| 连续学习7天 | +10 |

### 5. 奖励系统 🎁

**孩子可以"许愿"**：
> "小英，我要设置奖励！100分 = 买玩具！"

**家长可以"确认"**：
> "确认许愿" → 奖励生效

**奖励示例**：
| 积分 | 奖励 |
|------|------|
| 100分 | 去野餐 🎪 |
| 200分 | 买玩具 🧸 |
| 500分 | 周末旅游 ✈️ |

## 📊 数据沉淀配置（可选）

配置后可查看孩子的学习记录和进步趋势。

### 配置步骤

#### 1. 创建表格

飞书 → 多维表格 → 新建空白表格 → 命名为「学习记录」

#### 2. 添加列（按顺序添加）

| 列名 | 类型 | 选项 |
|------|------|------|
| 日期 | 日期时间 | - |
| 学科 | 单选 | 英语、数学 |
| 学习内容 | 文本 | - |
| 得分 | 数字 | - |
| 错误类型 | 文本 | - |
| 改进建议 | 文本 | - |
| 积分变化 | 数字 | - |
| 孩子状态 | 单选 | 正常、抵触、休息 |

#### 3. 获取表格 ID

打开表格 → 右上角「···」→ 复制链接

表格 ID 是链接中「base/」后面的那串字符

#### 4. 填入配置

在 Claw学伴后台 → 数据配置 → 粘贴表格 ID

---

> 💡 **不配置也能用**，数据会存在本地。配置后才会同步到飞书表格。
