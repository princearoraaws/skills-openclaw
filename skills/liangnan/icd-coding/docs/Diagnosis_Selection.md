# 主要诊断选择原则详解

> 本文档根据《病案信息学》及国家相关规范，详细阐述主要诊断选择原则

---

## 一、主要诊断的定义

**主要诊断**是指患者本次住院的主要原因，即导致患者住院的主要疾病或情况。

### 判断标准（三项原则）
1. **患者住院的主要原因**
2. **消耗医疗资源最多的疾病**
3. **影响患者康复的主要疾病**

> 在医保结算中，主要诊断直接影响DRG/DIP分组，必须准确选择。

---

## 二、主要诊断选择基本规则

### 1. 住院目的为主诊断
| 情况 | 主要诊断选择 |
|------|-------------|
| 因疾病住院 | 疾病本身 |
| 因并发症住院 | 并发症 |
| 因院内感染住院 | 院内感染 |

### 2. 疾病为主、手术/操作为辅
| 情况 | 主要诊断选择 |
|------|-------------|
| 住院行手术治疗 | 导致手术的疾病 |
| 住院行手术复查 | 疾病复查原因 |
| 住院行并发症治疗 | 并发症 |

### 3. 病因诊断优先
| 情况 | 主要诊断选择 |
|------|-------------|
| 疾病及其导致的其他疾病 | 病因 |
| 慢性病急性发作 | 慢性病 |
| 临床表现明确病因 | 明确病因 |

## 四、不应作为主要诊断的情况

### 1. 医院感染
- 住院期间获得的感染
- 除非该感染是患者住院的主要目的

### 2. 既往病史
- 既往病史
- 除非该病史是本次治疗的全部内容

### 3. 慢性疾病急性发作
- 慢性疾病的急性发作期
- 可作为其他诊断

### 4. 并发症 vs 合并症
- 并发症（因疾病或治疗导致）→ 可作为主诊断
- 合并症（患者存在的其他疾病）→ 作为其他诊断

---

## 五、其他诊断填写规范

### 其他诊断定义
其他诊断是指患者住院期间存在的、影响治疗或增加医疗资源的疾病或情况。

### 填写原则
1. **影响住院时间的诊断** → 必须填写
2. **影响治疗的诊断** → 必须填写
3. **增加医疗资源的诊断** → 必须填写
4. **既往病史** → 慢性病史应填写
5. **院感** → 住院期间发生的感染应填写

---

## 六、手术和操作填写规范

### 填写原则
1. **主要手术/操作** → 优先填写
2. **其他手术/操作** → 按手术大小/复杂程度依次填写
3. **手术方式** → 开放/腔镜/介入需区分
4. **手术部位** → 左/右侧需区分
5. **手术时间** → 需准确填写

---

## 七、DRG/DIP对编码的影响

### 主要诊断对DRG的影响
| 主要诊断 | DRG分组 |
|----------|---------|
| 心肌梗死 | MDC 5（循环系统疾病） |
| 肺炎 | MDC 4（呼吸系统疾病） |
| 胃癌手术 | 外科组 |

### 编码错误对DRG的影响
- 错误的主要诊断 → 进入错误的DRG组
- 错误的手术/操作 → 影响CMI值
- 漏填合并症/并发症 → 影响权重

### 常见编码问题
| 问题 | 影响 |
|------|------|
| 肿瘤原发/转移混淆 | DRG分组错误 |
| 糖尿病分型错误 | 进入错误ADRG |
| 损伤类型错误 | 分组错误 |
| 手术方式错误 | 手术权重错误 |

## 八、主要诊断选择流程图

```
患者入院
    ↓
确定住院目的（治疗/检查/手术等）
    ↓
确定主要疾病/情况
    ↓
判断是否符合主要诊断标准
    ↓
是否消耗最多医疗资源？
    ↓是/否
    ↓
是否为影响康复的主要疾病？
    ↓是/否
    ↓
选择主要诊断
```

## 九、常见问题解答

### Q1: 患者同时有多个疾病，如何选择主要诊断？
A: 选择消耗医疗资源最多、影响康复最大的疾病作为主要诊断。

### Q2: 手术后发现其他疾病，如何选择主要诊断？
A: 如果发现的疾病需要继续住院治疗，应作为主要诊断；否则作为其他诊断。

### Q3: 肿瘤患者复查如何选择主要诊断？
A: 复查无阳性发现 → 肿瘤复查；阳性发现 → 发现的疾病。

### Q4: 慢性病急性发作如何选择？
A: 一般选择慢性病作为主要诊断，急性发作作为其他诊断。

### Q5: 糖尿病患者入院调糖如何选择？
A: 以糖尿病调糖为目的 → 糖尿病；因糖尿病并发症入院 → 并发症。
