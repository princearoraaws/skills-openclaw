# color-palette-cn

配色方案生成、色彩和谐(互补/类似/三色)、品牌配色、对比度检查(WCAG)、颜色格式转换(HEX/RGB/HSL)、流行色推荐。Color palette generator with harmony, brand colors, WCAG contrast check, format conversion, trending colors.

**Q: 这个工具适合谁用？**
A: 任何需要color-palette-cn的人，无论是个人还是企业用户。

A: 主要输出Markdown格式，方便复制和编辑。

## 专业建议

- 法则** — 主色60%，辅色30%，点缀色10%
- 对比度很重要** — 文字至少4.5:1(AA级)，大文字3:1
- 不超过5种颜色** — 一个方案里3-5种色足够
- 先选主色** — 再用harmony命令生成搭配色
- `generate` — 不知道用什么色？给个关键词自动推荐
