# 淘股吧今日推荐抓取整理技能

## 描述
自动浏览淘股吧网站，进入"今日推荐"专栏，抓取排名前N的帖子，阅读完整内容后整理总结输出。

## 功能
- 自动导航到淘股吧首页
- 定位并切换到"今日推荐"专栏
- 逐个打开前N个帖子获取完整内容
- 整理总结：整体行情、板块表现、多数作者观点

## 使用场景
用户要求"浏览淘股吧今日推荐，整理前15个帖子观点"时使用。

## 操作步骤

### 方案一：浏览器方式（默认，可见浏览器）
适合调试和可视化验证

1. **打开浏览器导航**
```
browser action=start
browser action=navigate url=https://www.tgb.cn/
```

2. **确认"今日推荐"标签**
- 在首页导航标签中找到"今日推荐"
- 确认已经激活（如果没有激活点击切换）

3. **获取帖子列表**
- 从页面中提取前N个（通常是15个）帖子的链接
- 帖子链接格式为 `https://www.tgb.cn/a/xxxx`

4. **批量打开所有帖子**
- 提取到前N个帖子URL后，**同时打开所有帖子为独立标签页**（使用 `browser action=open url=...` 逐个调用）
- 并发加载所有帖子，节省等待时间

5. **逐个提取内容**
- 对每个打开的标签页，获取页面完整内容
- 提取作者、标题、主要观点

---

### 方案二：后台静默抓取（无浏览器窗口）
推荐用于定时任务，全程后台无界面

1. **抓取首页获取帖子列表**
```
web_fetch url=https://www.tgb.cn/
```

2. **提取帖子链接**
- 从HTML中解析"今日推荐"板块，提取前N个帖子URL
- 格式为 `https://www.tgb.cn/a/xxxx`

3. **并发抓取所有帖子**
- 对每个帖子URL调用 `web_fetch url=...` 批量获取内容
- 自动提取正文为markdown，不需要处理HTML

4. **提取作者、标题、主要观点**
- 直接从抓取的markdown内容中提取总结

两种方案都可以，优先使用**方案二（后台静默）**用于定时任务，不显示浏览器。

### 5. 整理总结
整理输出包含以下部分：
1. **整体总结** - 市场概况、成交量、整体情绪
2. **市场判断** - 趋势、板块表现（强弱分类）
3. **多数作者观点** - 操作策略共识、投资理念交流
4. **焦点个股** - 被多次提到的股票

## 注意事项
- 不需要登录也能浏览内容，不用登录
- 每个帖子只需要总结主要观点，不用整理评论区
- 保持客观，只整理作者表达的观点，不添加额外判断
- 输出格式清晰，使用markdown分级
